# Проект: Путешествие по России

### *О чем?*
Проект рассказывает и наглядно показывает возможности путешействий по России!

### *Какие технологии были использованы?*
* Flex-Box 
* Grid-Layout
* Адаптивная верстка под разные разрешения экрана
* Медиазапросы
Сайт был сверстан из макета редактора *Figma*
